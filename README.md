# Scripture Widgets

A personalized Bible verse widget app for Android.

## Features
- Daily Bible verse home screen widgets
- 15 verse categories (Hope, Faith, Peace, Healing, etc.)
- Personalized onboarding based on age, faith journey and goals
- Multiple notification schedules
- 12 widget themes with live preview
- Offline-first with 120+ pre-seeded verses

## Tech Stack
- Jetpack Compose
- Glance 1.1 (widgets)
- Room (local database)
- Hilt (dependency injection)
- WorkManager (notifications)
- DataStore (preferences)

## Setup
1. Clone the repo
2. Open in Android Studio
3. Add your Bible API key to local.properties: `BIBLE_API_KEY=your_key_here`
4. Run on device or emulator
